# 微信小程序开发者工具调试指南

## 微信小程序开发者工具使用方法

### 步骤 1：安装微信开发者工具

1. 访问微信公众平台下载微信开发者工具
2. 安装并完成微信开发者身份认证

### 步骤 2：导入项目并配置

1. 打开微信开发者工具
2. 选择"导入项目"，选择项目目录
3. 填写小程序AppID（测试可使用测试号）
4. 项目名称填写"KitchenMate"

### 步骤 3：使用开发者工具调试

1. **模拟器调试**：在模拟器中查看小程序运行效果
2. **调试器面板**：
   - Console：查看日志和错误信息
   - Sources：调试JavaScript代码
   - Network：监控网络请求
   - Storage：查看本地存储数据
   - AppData：实时查看页面数据变化
3. **真机调试**：通过二维码扫码在真机上调试

## 针对当前项目的注意事项

### 1. 环境配置
- 需要微信开发者工具最新版本
- 项目需要配置合法的AppID（可使用测试号）
- 确保网络连接正常，用于模拟网络请求

### 2. 调试技巧
- **WXML面板**：实时查看页面结构变化
- **AppData面板**：监控页面数据变化，便于调试数据绑定
- **断点调试**：在Sources面板设置断点进行代码调试

### 3. 常用调试功能

**实时预览：**
- 修改代码后自动刷新预览
- 支持热重载，提高开发效率

**性能分析：**
- 使用Audits面板分析小程序性能
- 查看页面渲染性能指标

**网络调试：**
- 监控所有网络请求
- 支持模拟网络环境（如弱网测试）

## 常见问题解决

### 1. 报错"app.json 文件读取失败"

检查 app.json 文件格式是否正确：

```json
{
  "pages": ["pages/index/index"],
  "window": {
    "navigationBarTitleText": "KitchenMate"
  }
}
```

### 2. 页面无法正常显示

检查页面路径配置：
- 确保 app.json 中 pages 数组包含正确的页面路径
- 页面文件应放在对应目录下
- 页面路径大小写敏感

## 总结

微信开发者工具是开发和调试原生小程序的主要工具。通过模拟器预览、调试器面板和真机调试功能，可以高效地开发和调试小程序应用。

建议在开发过程中充分利用调试器的各项功能，特别是AppData面板和网络监控，能有效提高开发效率。
